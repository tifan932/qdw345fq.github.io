    Netch 游戏加速 使用教程

    最后更新于 2021-09-19 17:24:32

    1.

    下载 Netch 客户端，解压后以管理员身份运行 Netch.exe。若系统提示需要安装 .NET Framework，请点此访问微软官网下载安装。

    下载客户端

    2.

    打开程序后，选中 “订阅” > “管理订阅链接”

    3.

    粘贴订阅链接到左下角的链接，备注填写为，点击添加，然后点击保存

    4.

    如果需要加速的游戏不在列表里面，那么就选中 “模式” > “创建进程模式”

    5.

    点击 “扫描” 选取你游戏安装目录后选择确定即可添加，点击保存即可选择应用此模式

    6.

    最后选择模式，点击启动即可使用。
