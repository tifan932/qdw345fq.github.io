常见问题

问 : 我买了应该怎么用？
答 :

可以参考我们的侧边栏使用教程，里面有各平台对应教程，移动端访问仅提供对应移动端平台教程，如按教程配置后依然无法使用，请联系我们在线客服进行指导

问 : 你们是什么网站？有什么用？
答 :

我们致力于帮助更多人通过我们的服务进行学习增长，资料搜集，国际社交。

问 : 你们的延迟怎样？
答 :

延迟会根据用户所在不同位置产生不同的物理距离延迟

问 : 我想注销账号应该怎么做？
答 :

在右侧导航 -> 我的账号中可以自行注销账号

但需要注意：注销账号后您的所有数据都会被永久删除，并且无法找回。

请谨慎操作！请谨慎操作！请谨慎操作！

问 : 为什么你们的IOS软件要收费？
答 :

我们统一采用市面常见并广泛应用的软件作为标准软件，由于使用人群大，在稳定性以及修复能力上都比自主开发强。所以开发者会收取一定的开发费用，此费用并非官方收取

问 : 为什么感觉到了晚上没有白天快？
答 :

到了晚间时刻，大家都回到了家，开始了各种的丰富晚间生活，这时候数据出口占用会相交白天压力要大，我们统称为晚高峰时段

问 : 审计规则是什么意思？
答 :

对于某些存在风险的网站类型，我们会进行审计屏蔽
