使用说明

节点筛选

线路简介：可以通过节点筛选来选择节点使用，这会更好的满足您的服务需求。

各类标签：流量倍率 -> 你使用 0.2 倍率的节点，消耗 1G 流量，则统计为 0.2G；速率 -> 标示速率为最高速率，但会受限于您本地网络、目标网络等；等级 -> 与会员套餐匹配；在线、负载 -> 表示节点当前使用人数，过载可能会影响线路体验...

使用权限：对应的节点使用权限根据你的用户等级和用户群组来决定的.
